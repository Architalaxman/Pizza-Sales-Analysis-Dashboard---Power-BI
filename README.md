# 🍕 Pizza Sales Analysis Dashboard - Power BI

## 📌 Project Overview
This project is an interactive **Pizza Sales Dashboard** built using Power BI.  
It provides detailed insights into sales performance, customer ordering patterns, and product trends.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify best-selling pizzas
- Track revenue trends over time
- Understand customer ordering behavior

---

## 📂 Dataset Features
The dataset includes:
- Order ID
- Pizza Name
- Category
- Size
- Quantity
- Price
- Order Date & Time

---

## 📊 Dashboard Insights
Key insights covered:
- 💰 Total Revenue & Sales Trends
- 🍕 Top & Bottom Selling Pizzas
- 📊 Category-wise Sales Analysis
- ⏰ Peak Order Time Analysis
- 📦 Size-wise Sales Distribution

---

## 🛠️ Tools & Technologies Used
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)

---

## 🚀 Features
- Interactive slicers and filters
- Dynamic charts and visuals
- Clean and user-friendly dashboard
- Drill-down insights
